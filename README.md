# Green-Guardian-AI 🌱

An AI-powered environmental assistant that helps users identify plant diseases, generate remedies, track recovery progress, and receive intelligent guidance using AI.

## Features

* 🤖 AI Chatbot for environmental and plant-related assistance
* 🌿 Plant disease analysis and recommendations
* 💊 Remedy generation based on detected issues
* 📈 Recovery tracking system
* 📄 Printable reports
* 🔥 Firebase integration
* ⚡ Fast React + TypeScript + Vite frontend

## Tech Stack

* React
* TypeScript
* Vite
* Node.js
* Firebase
* Gemini AI API

## Installation

```bash
git clone https://github.com/BayyaramHansika/Green-Guardian-AI.git
cd Green-Guardian-AI
npm install
```

## Environment Variables

Create a `.env` file in the project root:

```env
GEMINI_API_KEY=your_api_key_here
```

## Running the Project

```bash
npm run dev
```

The application will start at:

```
http://localhost:3000
```
